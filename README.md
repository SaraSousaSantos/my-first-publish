# Assistente de Aquariofilia - Possíveis Features:

Chatbot com IA (API) para dúvidas e dicas. Ex.: “Tenho guppies e plantas naturais, que filtro recomendas?”

Verificação de compatibilidade de peixes Base de dados + IA para explicar o motivo. Ex.: Guppies × Betta → Não compatíveis. Betta pode atacar guppies.

Base de dados de peixes, plantas, parâmetros da água e noções de setup

Lembretes de manutenção Ex.: Troca parcial de água

Calculadoras úteis Ex.: Volume do aquário, horas de luz para aquário plantado

Registo de vários aquários (população, parâmetros, medições e notas)

Comunicação entre utilizadores  (partilha de registos dos aquários com fotos)

Upload de foto para diagnóstico com IA (doenças, algas, problemas no aquário)

O Design deve ser fácil de navegar ( o utilizador muitas vezes tem as mãos molhadas ou esta a completar alguma tarefa).
As calculadoras não podem ser apenas conversores de unidades de medida.
As bases de dados tem que incluir plantas.
O Registo de Aquários devia incluir a quantidade dos vários produtos químicos (e quantidades) usados para manutenção.
